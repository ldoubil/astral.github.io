## ✨ 核心特性

- 🌐 **P2P 网络连接** - 基于 EasyTier 的去中心化网络架构
- 🔒 **安全加密传输** - 端到端加密，保护数据隐私
- 🖥️ **跨平台支持** - 支持 Windows、Android、iOS、macOS、Linux
- 🎨 **现代化界面** - 基于 Flutter 的美观直观用户界面
- ⚡ **高性能架构** - Rust 后端确保高效的网络处理
- 🔧 **简单易用** - 几分钟即可完成配置，轻松创建虚拟网络
