## 📊 服务器状态监控

### 实时状态指标

您可以通过以下方式检查服务器状态：

```bash
# 测试服务器延迟
ping public.easytier.top

# 检查端口连通性
telnet public.easytier.top 11010

# 使用 traceroute 查看路由
tracert public.easytier.top
```

## 🚨 服务器故障应对

### 服务器不可用处理

如果遇到服务器连接问题：

1. **🔄 切换服务器**
   - 立即更换到备用服务器
   - 通知团队成员同步切换

2. **📊 检查网络状况**
   - 测试本地网络连接
   - 检查是否为运营商问题

3. **⏰ 等待恢复**
   - 服务器维护通常在1-2小时内完成
   - 关注社区公告获取最新信息

4. **📞 反馈问题**
   - 在社区群组报告问题
   - 提供具体的错误信息

### 性能优化技巧

**🔧 配置优化**：
- 同时启用 TCP 和 UDP 协议
- 开启"延迟优先"模式
- 定期检查和更新配置

**🌐 网络优化**：
- 使用有线网络连接
- 关闭不必要的网络应用
- 优化本地网络设置
