---
title: 常见 NAT 类型及其解释
description: 各种 NAT 类型的含义与区别说明
---

NAT（网络地址转换）是一种让多台设备共享一个公网 IP 的技术。不同的 NAT 类型会影响网络通信，尤其是 P2P（点对点）应用。下面是常见的 NAT 类型及其简单解释：

---

| 序号 | NAT 类型                  | 含义说明                                                                 |
| :--: | :------------------------ | :----------------------------------------------------------------------- |
|  1   | 未知（Unknown）           | 无法检测到 NAT 类型，可能是网络异常或检测失败。                         |
|  2   | 开放网络（OpenInternet）  | 没有 NAT，设备直接连接互联网，通信最畅通。                               |
|  3   | 无PAT（NoPat）            | 没有端口转换，和开放网络类似，通信无障碍。                               |
|  4   | 全锥形（FullCone）        | 只要你主动连接过对方，对方就能随时联系你，适合大多数应用。               |
|  5   | 受限锥形（Restricted）    | 你主动联系过的 IP 才能联系你，安全性更高一点。                           |
|  6   | 端口受限锥形（PortRestricted） | 你主动联系过的 IP 和端口才能联系你，限制更严格。                    |
|  7   | 对称型（Symmetric）       | 每次连接外部都会用不同端口，P2P 连接最难，穿透难度最大。                 |
|  8   | 对称UDP防火墙（SymUdpFirewall） | 有防火墙阻挡 UDP 通信，P2P 连接受限。                             |
|  9   | 对称递增型（SymmetricEasyInc） | 端口号每次递增，穿透难度比普通对称型低一点。                      |
| 10   | 对称递减型（SymmetricEasyDec） | 端口号每次递减，穿透难度比普通对称型低一点。                      |


## 总结

- **开放网络/无PAT/全锥形**：最容易通信，P2P 友好。
- **受限锥形/端口受限锥形**：有一定限制，但大部分应用能正常用。
- **对称型/UDP防火墙/递增/递减**：限制多，P2P 连接难度大。
