# 🏠 创建房间

房间是 Astral 中的核心概念，它代表一个虚拟网络，连接到同一房间的设备可以相互通信。

## 🎯 什么是房间？

<CardGrid>
  <Card title="🌐 虚拟网络" icon="puzzle">
    每个房间都是一个独立的虚拟局域网
  </Card>
  <Card title="🔐 安全隔离" icon="shield">
    不同房间之间完全隔离，互不干扰
  </Card>
  <Card title="👥 多人协作" icon="star">
    多个设备可以同时连接到一个房间
  </Card>
  <Card title="🎮 即插即用" icon="rocket">
    连接后就像在同一个局域网中
  </Card>
</CardGrid>

## 📝 创建新房间

<Steps>
1. **进入房间管理**
   - 点击左侧菜单的**"房间"**选项

2. **创建新房间**
   - 点击**"加号"**图标 ➕

3. **设置房间信息**
   - **房间名称**：给房间起一个便于记忆的名字
   - **房间号**：设置一个唯一的房间标识符
   - **房间密码**（可选）：为房间设置密码保护

4. **确认创建**
   - 点击**"确定"**完成创建
</Steps>

![创建房间](/images/食用-5.png)

## 🔧 房间配置详解

### 房间名称
- 📝 **作用**：帮助您识别不同的房间
- 💡 **建议**：使用有意义的名称，如"MC服务器"、"工作网络"等
- ✅ **示例**：`我的游戏房间`、`团队协作网络`

### 房间号
- 🔑 **作用**：房间的唯一标识符，朋友通过此号码加入房间
- 📏 **格式**：通常是数字或字母组合
- ✅ **示例**：`12345`、`game2024`、`work-room`

### 房间密码
- 🔒 **作用**：防止未授权用户加入房间
- 🆔 **可选性**：可以留空，创建公开房间
- 💡 **建议**：如果有隐私需求，建议设置密码

## 🛡️ 房间保护模式

![房间保护](/images/食用-8.png)

**保护房间**功能提供额外的安全保护：

- 🔐 **加密房间信息**：隐藏房间号和密码
- 📱 **分享码访问**：只能通过专用分享码加入
- 🚫 **防止暴力破解**：提高房间安全性

<Aside type="tip" title="何时使用保护模式？">
- 处理敏感信息时
- 限制访问人员时  
- 需要更高安全性时
</Aside>

## 🔗 连接到房间

创建房间后，您需要连接到它：

<Steps>
1. **选择房间**
   - 在房间列表中点击您刚创建的房间
   - 确保房间呈现**选中状态**

2. **建立连接**
   - 返回主界面
   - 点击**"连接"**按钮

3. **确认连接状态**
   - 等待连接建立
   - 网络状态显示**"已连接"**
</Steps>

![连接房间](/images/食用-6.png)

## 📋 查看房间状态

连接成功后，返回房间页面，您应该能看到：

- 🖥️ **本机**：显示您的设备信息
- 🌐 **服务器**：显示连接的中继服务器
- 📊 **连接状态**：显示网络连接质量
- 🏷️ **IP地址**：显示在虚拟网络中的IP地址

![房间状态](/images/食用-7.png)

<Aside type="note" title="关于 NAT 类型">
本机下方显示的"端口受限锥型"等信息是您的 NAT 类型。不同的 NAT 类型会影响连接质量，详情请查看 [NAT 类型说明](../../troubleshooting/nat-types.mdx)。
</Aside>

## 📤 邀请朋友

### 方式一：分享房间信息
告诉朋友：
- 🔢 **房间号**：他们需要输入这个号码
- 🔐 **房间密码**：如果您设置了密码
- 🌐 **服务器地址**：确保选择相同的服务器

### 方式二：使用分享码（推荐）
如果启用了房间保护：
1. 在房间页面点击**"复制分享码"**
2. 将分享码发送给朋友
3. 朋友使用分享码即可直接加入
