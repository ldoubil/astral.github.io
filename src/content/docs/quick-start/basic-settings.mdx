# ⚙️ 基础设置

完成安装后，让我们进行一些基础设置，确保 Astral 能够正常工作。

## 🔥 重要：关闭防火墙

<Aside type="caution" title="必须执行">
**这是最重要的步骤！** 大部分无法联机的问题都是由防火墙引起的。
</Aside>

### Windows 防火墙设置

<Steps>
1. 打开 Windows 设置（Win + I）
2. 选择"更新和安全" → "Windows 安全中心"
3. 点击"防火墙和网络保护"
4. **临时关闭**专用网络和公用网络的防火墙
</Steps>

![关闭防火墙](/images/食用-1.png)

### 为什么需要关闭防火墙？

- 🚫 **阻止连接**：防火墙会拦截 P2P 网络的连接请求
- 🔒 **端口限制**：限制应用使用必要的网络端口
- ⚡ **影响性能**：可能导致网络延迟增加

<Aside type="note">
在熟悉 Astral 的网络需求后，您可以配置防火墙规则来允许特定端口，而不是完全关闭防火墙。
</Aside>

## 🏷️ 设置用户名称

启动 Astral 应用后：

<Steps>
1. 在主界面找到**名称设置**区域
2. 输入一个**唯一的用户名**
3. 确保名称符合以下要求：
   - ✅ 使用英文字母和数字
   - ✅ 避免特殊符号
   - ✅ 与其他用户不重复
</Steps>

![用户名设置](/images/食用-6.png)

<Aside type="caution" title="命名规则">
- ❌ 不要使用中文字符
- ❌ 不要使用特殊符号
- ❌ 不要与房间内其他用户重名

违反这些规则可能导致无法连接或功能异常。
</Aside>

## 📡 选择服务器

<Steps>
1. 点击左侧菜单的**"服务器"**选项
2. 点击右下角的**地球图标**🌍
3. 从列表中选择一个服务器
4. 点击**"启用"**按钮
5. 点击**"编辑"**，确保勾选 **TCP** 和 **UDP** 协议
</Steps>

![服务器选择](/images/食用-2.png)
![服务器启用](/images/食用-4.png)

### 如何选择服务器？

- 🏃‍♂️ **延迟优先**：选择延迟最低的服务器
- 🌍 **地理位置**：选择离您最近的服务器
- 👥 **与朋友同步**：确保所有用户选择相同的服务器

## 🔧 网络优化设置

<Steps>
1. 进入**"设置"**页面
2. 找到**"网络设置"**
3. 开启**"延迟优先"**选项
</Steps>

![网络设置](/images/食用-5.png)

这个设置可以：
- ⚡ 优化网络路由
- 📈 减少连接延迟
- 🎯 提升游戏体验

## ✅ 验证设置

完成以上设置后，检查：

- [ ] 防火墙已关闭
- [ ] 用户名已设置且符合规范
- [ ] 服务器已选择并启用
- [ ] TCP/UDP 协议已启用
- [ ] 延迟优先已开启
