完成安装后，让我们进行一些基础设置，确保 Astral 能够正常工作。

## 🔥 重要：关闭防火墙

<Aside type="caution" title="必须执行">
**这是最重要的步骤！** 虽然这并不好，但是大部分无法联机的问题都是由防火墙引起的。
</Aside>

### Windows 防火墙设置

<Steps>
1. 打开 Astral 软件
2. 点击下图红色方框中的按钮，**临时关闭**防火墙
</Steps>

![关闭防火墙](/images/食用-1.png)

### 为什么需要关闭防火墙？

- 🚫 **阻止连接**：防火墙会拦截 P2P 网络的连接请求
- 🔒 **端口限制**：限制应用使用必要的网络端口
- ⚡ **影响性能**：可能导致网络延迟增加

<Aside type="note">
在熟悉 Astral 的网络需求后，您可以配置防火墙规则来允许特定端口，而不是完全关闭防火墙。
</Aside>

## 🏷️ 设置用户名称

启动 Astral 应用后：

<Steps>
1. 在主界面找到**用户信息**区域
2. 输入一个**唯一的用户名**
3. 确保名称符合以下要求：
   - ✅ 与其他用户不重复
   - ✅ 能区分出房主和房员
</Steps>

![用户名设置](/images/食用-6.png)

<Aside type="caution" title="命名规则">
- ❌ 不要与房间内其他用户重名

违反这规则可能导致无法连接或功能异常。
</Aside>

## 📡 选择服务器

<Steps>
1. 点击左侧菜单的**"服务器"**选项
2. 点击右下角的**地球图标**🌍

![服务器选择](/images/食用-2.png)

3. 选择列表中最上方的三个服务器
4. 点击**"启用"**按钮

![服务器启用](/images/食用-4.png)

5. 点击“启用”按钮右边的**"编辑服务器"**，确保勾选 **TCP** 和 **UDP** 协议
</Steps>

<Aside type="caution" title="注意">
- UDP协议需要服务器支持
- 该设置仅对转发模式有效
</Aside>

![服务器调整](/images/食用-10.png)



### 如何选择服务器？

- 🏃‍♂️ **延迟优先**：选择延迟最低的服务器
- 🌍 **地理位置**：选择离您最近的服务器

<Aside type="caution" title="必须相同">
👥 **与朋友同步**：确保所有用户选择相同的服务器，否则无法正常连接。
</Aside>

## 🔧 网络优化设置

<Steps>
1. 进入**"设置"**页面
2. 找到**"网络设置"**
3. 开启**"延迟优先"**选项
</Steps>

![网络设置](/images/MC联机/MC联机-15.png)

这个设置可以：
- ⚡ 优化网络路由
- 📈 减少连接延迟
- 🎯 提升游戏体验

## ✅ 验证设置

完成以上设置后，检查：

- [ ] 防火墙已关闭
- [ ] 用户名已设置且能区分房主和房员
- [ ] 服务器已选择并启用
- [ ] TCP/UDP 协议已启用
- [ ] 延迟优先已开启
