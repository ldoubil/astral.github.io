## 安装说明

### Windows
- 下载 `.exe` 安装包双击安装
- 或下载便携版 `.zip` 解压后运行

### Android
- 下载 `.apk` 文件
- 允许安装未知来源应用
- 点击安装包完成安装

### Linux
- 下载对应发行版的安装包
- 或下载 AppImage 添加执行权限后运行

## 下一步
