## 安装说明

### Windows
- 下载 `.exe` 安装包后双击安装
- 需要便携版时下载 `.zip` 解压后直接运行

### Android
- 下载 `.apk` 文件
- 在系统设置中允许安装未知来源应用
- 点击安装包完成安装

### Linux
- 下载对应发行版的安装包（deb/rpm 等）
- 或下载 AppImage，添加执行权限后运行
