**Astral** 是基于 [EasyTier](https://easytier.cn) 的跨平台 P2P 组网工具，用更直观的界面把复杂的虚拟局域网搭建流程简化到几分钟。
它既适合游戏联机，也适合小团队共享内部服务，帮助你在安全的网络中快速互联。

## 🎯 主要应用场景

### 🎮 游戏联机
- **多人开黑**：与朋友快速搭建私有房间，局域网游戏也能异地畅玩
- **稳定低延迟**：智能路由与穿透优化，降低卡顿和掉线
- **私密可靠**：只邀请成员加入，保障游戏数据与语音安全

## 🛠️ 技术特点

- **基于 EasyTier**：采用成熟的网络穿透技术
- **Flutter 界面**：轻量且现代化的用户体验
- **Rust 后端**：高性能与内存安全并重
- **NAT 穿透**：自动适配复杂网络环境
- **智能路由**：自动选择最优连接路径
