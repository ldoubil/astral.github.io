# 🚀 什么是 Astral？

**Astral** 是一个基于 [EasyTier](https://easytier.cn) 的现代化跨平台网络应用，旨在简化 P2P 网络连接和虚拟专用网络的创建与管理。

## ✨ 核心优势

<CardGrid>
  <Card title="🌐 去中心化网络" icon="star">
    基于 P2P 技术，无需依赖中央服务器，直接连接设备
  </Card>
  <Card title="🔒 安全可靠" icon="shield">
    内置加密传输，保护您的数据安全和隐私
  </Card>
  <Card title="📱 跨平台支持" icon="laptop">
    支持 Windows、Android、iOS、macOS、Linux 等主流平台
  </Card>
  <Card title="⚡ 简单易用" icon="rocket">
    现代化界面设计，几分钟即可完成配置
  </Card>
</CardGrid>

## 🎯 主要应用场景

### 🎮 游戏联机
- **Minecraft 多人游戏**：与朋友轻松搭建私人服务器
- **局域网游戏**：突破地理限制，异地联机如同在身边
- **低延迟体验**：智能路由优化，减少网络延迟

### 🏠 智能家居
- **设备管理**：远程控制家中的智能设备
- **网络监控**：实时查看家庭网络状态
- **安全访问**：加密连接，保护隐私安全

### 📁 个人文件共享
- **家庭网络**：在不同设备间安全传输文件
- **个人存储**：访问家中的 NAS 或个人服务器
- **朋友间分享**：临时创建网络进行文件传输

## 🛠️ 技术特点

- **基于 EasyTier**：采用成熟的网络穿透技术
- **Flutter 界面**：现代化的用户体验
- **Rust 后端**：高性能和内存安全
- **NAT 穿透**：自动处理复杂的网络环境
- **智能路由**：自动选择最优连接路径
