# 🚀 Astral - 轻松驾驭您的虚拟网络

Astral 是一个创新的**跨平台网络应用**，旨在为用户提供**简单易用的 P2P 网络连接和 VPN 服务**。借助现代化、直观的用户界面，Astral 让您能够轻松创建、管理并掌控您的虚拟网络。


---
## ✨ 主要特性

- 🌐 **P2P 网络连接** - 基于 EasyTier 的去中心化网络架构
- 🔒 **VPN 服务** - 安全的虚拟专用网络连接
- 🖥️ **跨平台支持** - 支持 Windows、macOS、Linux、Android 和 iOS
- 🎨 **现代化界面** - 基于 Flutter 的美观用户界面
- ⚡ **高性能** - Rust 后端确保高效的网络处理
- 🔧 **易于配置** - 简单的房间和服务器管理
